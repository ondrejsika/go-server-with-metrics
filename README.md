# go-server-with-metrics

Simple Go server with `/metrics` endpoint for Prometheus. Available on Docker Hub `ondrejsika/go-server-with-metrics`

## Build & Push to Docker Hub

```
make
```

Or

```
make build
make push
```

## Run Locally

```
make run
```

See:

- http://127.0.0.1
- http://127.0.0.1/metrics

### Stop

```
make stop
```
